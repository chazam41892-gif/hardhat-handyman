## 📸 Image & Video Guidelines

**IMPORTANT:** Only use these formats:
- ✅ **Images:** `.jpg`, `.jpeg`, `.png`, `.webp`
- ✅ **Videos:** `.mp4`, `.webm`
- ❌ **NOT ALLOWED:** `.heic`, `.heif`, `.hevc`, `.mov` (iPhone formats)

**Why?** iPhone's default formats (HEIC/HEVC) don't work on websites. They must be converted to JPG/MP4 first.

**How to convert:**
- On iPhone: Settings → Camera → Formats → Select "Most Compatible"
- Already have HEIC files? Use https://heictojpg.com/ (free)
- Already have MOV/HEVC videos? Use https://cloudconvert.com/ (free)

See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed instructions.
